WebVR API (navigator.getVRDisplays)

Client-side JavaScript API for querying connected VR headsets and peripherals. Returns a Promise that resolves an array of available VRDisplays.

Samsung Internet Browser for GearVR

Somewhat supported

  • Chrome Supported in experimental builds. Support is disabled by default ("Enable VR" flag toggles support; load chrome://flags/#enable-webvr and load chrome://flags/#enable-gamepad-extensions). Read Chromium instructions.
  • Firefox Supported in Firefox Nightly. Read Firefox Nightly instructions.
  • Samsung Internet Browser for GearVR Experimental, WIP implementation in Samsung Internet Browser for Gear VR can be enabled by loading the URL internet://webvr-enable. (Note: performance is actively being worked on to reduce latency.)

    Web Speech API (speech recognition)

    The Speech Recognition part of the API provides speech-input features. No keyboards in VR!

    Spec. Docs.

    Chrome

    Supported since version 25

    Firefox

    Somewhat supported since version 45

    Opera

    Somewhat supported since version 27

    Safari

    No support

    Edge

    No support

    Samsung Internet Browser for GearVR

    Supported

    • Chrome Requires -webkit prefix, some attributes missing. See notes on Chromium.
    • Firefox Must be enabled under about:config by changing media.webspeech.recognition.enabled to true.
    • Opera Requires -webkit prefix; some attributes missing.
    • Safari No public signals yet.
    • Edge In development.
    • Samsung Internet Browser for GearVR Also has (proprietary) on-screen keyboard for text entry.
