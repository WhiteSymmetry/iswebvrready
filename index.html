WebVR API (navigator.getVRDisplays)

Client-side JavaScript API for querying connected VR headsets and peripherals. Returns a Promise that resolves an array of available VRDisplays.

Samsung Internet Browser for GearVR

Somewhat supported

  • Chrome Supported in experimental builds. Support is disabled by default ("Enable VR" flag toggles support; load chrome://flags/#enable-webvr).
  • Firefox Supported in Firefox Nightly.
  • Samsung Internet Browser for GearVR Experimental, WIP implementation in Samsung Internet Browser for Gear VR can be enabled by loading the URL internet://webvr-enable. (Note: performance is actively being worked on to reduce latency.)
