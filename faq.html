Can the WebVR API be polyfilled?

There are certain features that are possibly only by making changes to the browser platform. The most significant would the ability to traverse links.

When and how will link traversal work?

Link traversal is functional in Chromium builds and soon Firefox. Documentation on sample usage is coming soon.

Is the spec on the standards body track?

We have formed a W3C Community Group, and the spec lives in a W3C repo. Standardization of the spec has not begun yet.

Can I use HTML and CSS for WebVR?

This is not currently possible. However, there has been experiments that would enable the copy of DOM element layouts into textures usable in a webGL scene. Though, this will be quite a ways out before we see anything usable by regular web content.

What is WebVR?

WebVR is a set of APIs that offer the ability to render and interact with web content in VR.

Why WebVR?

Most VR content today is being created using proprietary closed-source software. WebVR enables you to build VR experiences using web technologies such as WebGL and utilize the entire bredth of APIs already available in browsers. The Web also offers accessibility by URL, with no download and installs required and completely open platform for distributing your application.

What are the easiest ways to get started building WebVR experiences?

A-Frame, webvr-boilerplate, three.js.

Is WebVR supported on mobile devices?

Firefox for Android currently has WebVR APIs enabled by default. For browsers that do not support WebVR APIs natively, you can polyfill the behavior using the webvr-polyfill.

Which browsers support WebVR?

Currently, Firefox Nightly, experimental builds of Chromium, and experimental support in Samsung Internet for Gear VR.

Can I use WebVR on Mac or Linux?

The recommended platform for using WebVR is Windows using the Oculus Rift VR headset, or using your mobile device with a Cardboard holder. See section above.

Although using WebVR on Mac OS X and Linux is possible using the Oculus 0.5.0 runtime, it is not recommended. Oculus has suspended support for Mac/Linux as of since Oculus 0.5.0 runtime and it is unclear when support will be resumed.

How do I use WebVR with Samsung Gear VR?

If you haven’t already, insert your Samsung Gear VR-compatible phone into the Gear VR headset. This will download and install the Oculus Home Android app.

Experimental WebVR support for Gear VR can be enabled in the Samsung Internet Browser for Gear VR (which can be downloaded from the Oculus Home app). In the Browser, you can enable WebVR support by loading the URL internet://webvr-enable and disable support with the URL internet://webvr-disable.

It’s worth noting that the Samsung Internet Browser’s WebVR performance is actively being worked on to reduce current latency. Expect noticeable improvements soon.
